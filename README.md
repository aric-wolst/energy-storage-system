# Setup Instructions

## Backend

1. Have Node installed https://nodejs.org/en/download/

2. ```cd backend```

3. ```npm i```

4. Create ```.env``` file from ```.env.template```

5. ```npm start```

## Frontend

1. Have Node installed https://nodejs.org/en/download/

2. ```cd frontend```

3. ```npm i```

4. ```npm start```

5. Go to ```http://localhost:3000/``` on your browser
